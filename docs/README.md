## Обзор проекта

**Бирка** — Telegram Mini App для управления фулфилментом:

- **Клиенты** — просмотр статусов заказов, остатков, фото бракованных позиций.
- **Склад** — приёмка, упаковка, отгрузка заказов.
- **Бот** — ответы пользователям с помощью AI (заявки, товары, остатки, брак, отгрузки, прайс, реквизиты).

---

## Архитектура

- **Frontend** — SPA (React + Vite), статика через Nginx.
- **Backend** — REST API (FastAPI), префикс `/api/v1`. Авторизация: `X-Telegram-Init-Data` или `X-Session-Token`.
- **PostgreSQL** — заказы, пользователи, компании, товары, услуги, отгрузки, чат AI, RAG.
- **S3** — в БД только ключ объекта; URL строится на бэкенде. После загрузки — HEAD-проверка.
